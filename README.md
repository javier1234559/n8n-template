# n8n Workflows

This repo holds many **unrelated** n8n workflows. Each workflow is stored by project and environment.

## Setup

1. Run the script to start the n8n-mcp server

```bash
chmod +x run.sh
./run.sh
```

2. @AGENT.md to build the workflow

Conventions (project structure, file naming, per-project README) are in **docs/** — see [docs/CONVENTION_PROJECT_STRUCTURE.md](docs/CONVENTION_PROJECT_STRUCTURE.md).
